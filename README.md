This project is made with the help of AI tools like claude, trae, etc. It is fully vibecoded and i created this project to prepare for the upcoming DataForGood Hackathon.
The feature used in this project are:
- 🔍 Search weather by city name
- 📍 Auto-detect location using browser Geolocation API
- 🌡 Current weather: temperature, humidity, wind speed, weather condition
- ⏰ Hourly forecast: next 24 hours in a scrollable row
- 📅 5-day forecast: daily min/max temperature with weather icons
- 🌙 Dark mode: toggle with localStorage persistence
- 🔁 °C / °F toggle: switch units without re-fetching
- 🎨 Dynamic background: changes based on weather condition
- 🕐 Recent searches: last 5 cities saved locally
- ⚠️ Weather alerts banner: shows severe weather warnings
- 🏙 Compare two cities: side-by-side weather comparison
- Anamoly detection is added
